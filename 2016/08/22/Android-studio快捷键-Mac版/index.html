Android studio快捷键(Mac版)

2016-08-22
工具
Android studio
快捷键 功能
Control + Space 自动完成代码
Control + Shift + Space 智能自动完成代码
Command + Shift + Enter 完成声明, 如果最后没有分号, 添加分号, 如果有分号, 则换行
Control + P 查看方法参数
Shift + Cmd + F7 代码高亮
Shift + Cmd + G 代码高亮向上查找
Cmd + G 代码高亮向下查找
Cmd + F12 文件方法结构
Ctrl + Alt + H 查找调用的位置
Cmd + Space 快速查找定义
Cmd + Plus,Minus 折叠展开代码块
Cmd + Shift + Plus,Minus 折叠展开全部代码块
F3 添加书签
Alt + F3 添加书签标识
Cmd + Shift + A 查找动作
Alt + Shift + Up/Down 上下移动代码
Cmd + Delete 删除行
Cmd + D 快捷向下复制行
Alt + Up/Down 扩大缩小选中范围
Ctrl + O 快捷覆写方法
Cmd + Left/Right 快捷定位到行首/尾
Cmd + J 按照模板生成代码
Cmd + / 注释代码(//)
Cmd + Alt + / 注释代码(/**/)
Cmd + Alt + L 格式化代码
Alt + F1 快捷选择
Ctrl + Up/Down 快捷移动
Ctrl + Shift + J 快捷退至上一行末尾，取消换行
Cmd + E 快捷最近打开
Alt + Control + O 清楚无效引用
Cmd + Shift + Enter 代码补全
Alt + Enter 提示错误解决方案
Cmd + P 提示参数类型
Cmd + R 查找+替换
Cmd + F 查找
Ctrl + Option + H 查找调用的位置
Cmd + Shift + U 大小写转换
